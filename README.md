## 기본 환경

1. **JDK 17**
2. **Spring Boot 3.3.4**
3. H2 데이터 베이스

## 주요 의존성

1. **Spring Web**
2. **Thymeleaf**
3. **Spring Data JPA**
4. **H2 Database**
5. **Lombok**
6. **Swagger**(`http://localhost:8080/swagger-ui.html`)

# 도메인 분석 설계

## 요구사항 분석

1. 메뉴기능
    - 메뉴 목록 조회
    - 메뉴 추가 기능 / 메뉴 수정 (주문 가능 여부, 옵션)
    - 메뉴 삭제
    - 메뉴 복구
    - 카테고리 등록
    - 카테고리 수정
    - 카테고리 삭제
    - 카테고리 조회
2. 주문기능
    - 메뉴에서 항목 추가(옵션 선택 (예: 샷 추가, 시럽 추가, ice/hot), 수량)
    - 주문 항목중 선택항목 삭제 - 보류
    - 주문 취소
    - 주문 내역 확인
3. 주문처리 기능
    - 주문 내역 확인
    - 준비완료 처리
